Introduction
============

This product defines the *favicon* (also known as *favourite
icon,* *page icon,* *webpage icon,* and *shortcut icon*) for
GroupServer.

It provides a viewlet_, a page_, and a `resource directory`_, all
of which contain `the icon`_ itself.
